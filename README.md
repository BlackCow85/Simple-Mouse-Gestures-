<img width="284" height="285" alt="image" src="https://github.com/user-attachments/assets/f526e627-f69e-4f31-a12d-3f66156c9fbb" /># Simple Mouse Gestures (간편 마우스 제스처)

크롬 브라우저를 위한 가볍고 직관적인 마우스 제스처 확장 프로그램입니다.
복잡한 설정 없이 필수적인 탐색 기능들을 마우스 오른쪽 드래그만으로 빠르게 실행할 수 있습니다.

## ✨ 주요 기능

웹페이지 어디서든(광고 영역 포함) 마우스 오른쪽 버튼을 누르고 드래그하면 제스처가 인식됩니다. 드래그하는 동안 보라색 선으로 궤적이 표시됩니다.

### 제스처 목록

| 제스처 방향 | 동작 | 설명 |
| :--- | :--- | :--- |
| **←** (왼쪽) | **뒤로 가기** | 이전 페이지로 이동합니다. |
| **→** (오른쪽) | **앞으로 가기** | 앞 페이지로 이동합니다. |
| **↑** (위) | **맨 위로** | 현재 페이지의 최상단(Home)으로 스크롤합니다. |
| **↓** (아래) | **맨 아래로** | 현재 페이지의 최하단(End)으로 스크롤합니다. |
| **↓↑** (아래 갔다 위로) | **새로고침** | 현재 탭을 새로고침합니다. |
| **↑↓** (위 갔다 아래로) | **탭 닫기** | 현재 보고 있는 탭을 닫습니다. |
| **↑→** (위 갔다 오른쪽) | **닫힌 탭 열기** | 가장 최근에 닫은 탭을 다시 엽니다. (ㄱ 반대 모양) |

## 🚀 설치 방법 (개발자 모드)

이 확장 프로그램은 현재 크롬 웹 스토어에 등록되지 않았으므로, 수동으로 설치해야 합니다.

1. 이 저장소(Repository)의 파일을 모두 다운로드하여 한 폴더에 압축을 풉니다. (이미지 생성기 `icon_maker.html`은 제외해도 됩니다.)
    * 필수 파일: `manifest.json`, `background.js`, `content.js`, `popup.html`, `icon.png`
2. 크롬 브라우저 주소창에 `chrome://extensions` 를 입력하여 이동합니다.
3. 우측 상단의 **'개발자 모드'** 스위치를 켭니다.
4. 좌측 상단에 나타난 **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭합니다.
5. 1번에서 다운로드한 파일들이 들어있는 폴더를 선택합니다.
6. 설치가 완료되면 주소창 옆 퍼즐 아이콘을 눌러 확장 프로그램을 고정해두면 편리합니다.

## ⚠️ 주의 사항

* **작동 제한 페이지:** 크롬 브라우저의 보안 정책상 `chrome://` 설정 페이지, 웹 스토어 페이지, 그리고 아무 내용이 없는 '새 탭' 페이지에서는 제스처가 작동하지 않습니다. 일반 웹사이트(네이버, 구글 등)에서 테스트해주세요.
* **시크릿 모드:** '닫힌 탭 다시 열기' 기능은 시크릿 모드에서는 작동하지 않습니다.

## 🛠️ 기술 스택

* Chrome Extension Manifest V3
* Vanilla JavaScript (No external libraries)
